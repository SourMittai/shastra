# Shastra
Pronounced /ʃɑːs.t̪ɾᵊ/ or śāstra, Shastra is a simple Physics simulation engine which, while the starting is simple, it aims to develop into a more detailed physics simulator. In Sanskrit, Shastra means science or knowledge, and since this simulator is a free to use, 'unlicenced' simulator, Shastra was the name which was picked.

The whole software follows the Unlicense license so you are basically free to do anything you want with this code. The application is made by an aspiring astrophysicist going to IB DP1, and the idea of a physics simulator is given by Gemini AI.
